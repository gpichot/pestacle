## **Just MDX** {morph="section-title"}

Slides are separated by `---`.
Add a layout via frontmatter.
Drop in components when Markdown isn't enough.

```mdx
# Hello

<MyComponent />
```

---

layout: centered

---

import { HorizontalList, HorizontalListItem } from "@gpichot/spectacle-deck";

### **Three steps** {morph="section-title"}

<HorizontalList columns={3}>
  <HorizontalListItem title="1. Write">

Author your slides in `.mdx` files with familiar Markdown syntax.

  </HorizontalListItem>
  <HorizontalListItem title="2. Configure">

Pick a theme and layouts in `pestacle.config.ts`.

  </HorizontalListItem>
  <HorizontalListItem title="3. Present">

`pnpm dev` — navigate with arrow keys, present in fullscreen.

  </HorizontalListItem>
</HorizontalList>
