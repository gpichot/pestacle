---
title: section
description: A title-card layout for section dividers.
---

The `section` layout is designed for section title cards — a centered, bold
statement to introduce a new topic.

## Usage

```mdx
---
layout: section
---

# **New Topic** {morph="section-title"}

A brief description of this section. {morph="section-desc"}
```

## Behavior

- Content is centered and styled as a section divider
- Pairs well with morph directives for smooth transitions between section cards
  and content slides
