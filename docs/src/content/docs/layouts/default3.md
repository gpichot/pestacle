---

The `default3` layout features decorative hexagonal patterns on one side and
your content on the other.

## Usage

```mdx
---
layout: default3
position: right
---

# Layout Title

Content here, with hexagons on the other side.
```

## Frontmatter Props

| Prop       | Type                  | Default | Description                                  |
| ---------- | --------------------- | ------- | -------------------------------------------- |
| `position` | `"left"` \| `"right"` | —       | Which side the decorative hexagons appear on |

## Behavior

- Hexagonal patterns provide visual interest on one side
- Content occupies the other side
- Useful for slides that need a more visually dynamic background without a full
  image
